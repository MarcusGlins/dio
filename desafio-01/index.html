Introdução

(Voltar)

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. Ela serve para criar e organizar o conteúdo de um site, usando tags que definem como cada elemento deve aparecer ou funcionar na página.

Essa linguagem não é usada para criar lógica ou algoritmos, mas para estruturar e formatar documentos que podem ter links para outros documentos ou recursos na web. (Fonte: developer.mozilla.org)

Características básicas da Linguagem

(Voltar)

HTML é composta por elementos chamados de tags, que definem como o conteúdo deve ser apresentado no navegador. No exemplo abaixo temos a tag p, que informa ao navegador que o conteúdo do texto deverá apresentado em formato de parágrafo.

Exemplo: tag p 

      <p>
        Este texto será exibido em formato de parágrafo, portanto, terá uma delimitação específica deste conteúdo dentro 
da página web.
      </p>

Atributos das Tags HTML

(Voltar)

As tags HTML podem receber configurações adicionais. Através dos atributos vinculados à tag é possível refinar a formatação do conteúdo a ser exibido.

Exemplo: atributo id 

      <p id="paragrafo-principal">
        Este parágrafo recebe o atribudo id com o valor "paragrafo-principal", que identifica e o prepara para outras formatações.
      </p>

Textos

(Voltar)

Em HTML exitem algumas formas de se formatar textos, pois a linguagem foi criada com propósito de estruturar textos científicos e exibi-los para comunicação.

Inicialmente, temos a formatação dos títulos que possui uma hierarquia que hoje tem sido utilizada principalmente para auxiliar leitores de tela na busca de informações. Para exibir um título em HTML utilizamos as tags que são enumeradas de <h1> a <h6>, sendo o primeiro considerado o título principal da página e o segundo o último nível de subtítulos. Cada nível de título também possui um tamanho de fonte a ser exibido.

Exemplo: 

      <h1>Título Principal</h1> Possui o maior tamanho de fonte
      

      <h2>Subtítulo</h2> Tamanho de fonte menor que h1
      

      <h3>Subtítulo</h3> Tamanho de fonte menor que h2
      

      <h4>Subtítulo</h4> Tamanho de fonte menor que h3
      

      <h5>Subtítulo</h5> Tamanho de fonte menor que h4
      

      <h6>Subtítulo</h6> Possui o menor tamanho de fonte

A tag p apresentada anteriormente é um dos elementos mais utilizados, servindo com delimitador dos parágrafos.

Exemplo: 

      <p>
        Este é o primeiro parágrafo.
      </p>
      <p>
        Este é o segundo parágrafo.
      </p>

A citação de textos científicos é algo muito comum nesse gênero, e como herança dessa primeira finalidade da linguagem temos tag para formatar citações. Que exibe um recuo e destaca o texto na página.

Exemplo: 

      <blockquote>
        Quando acordei hoje de manhã, eu sabia quem eu era, mas acho que já mudei muitas vezes desde então.
      </blockquote>

Listas

(Voltar)

Assim como a formatação de textos é importante, a sumarização de um conteúdo facilita a busca pela informação mais rapidamente. Para isso, a linguagem conta com o recurso das listas. Que podem ser apresentadas em dois formatos: de listas ordenadas (que são estruturadas com a tag ol) e listas não-ordenadas (que são estruturadas com a tag ul)

As listas ordenadas exibem um número de identificação de cada item automaticamente, o que facilita a hierarquização da informação.

Exemplo: 

      <ol>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
      </ol>

      Será exibido:
  1. Item 1
  2. Item 2
  3. Item 3

As listas não-ordenadas não possuem um número de identificação, que hierarquiza a informação. Mas, que pode servir para apresentar a divisão de tópicos (Quando estamos falando de textos, é claro! Sua utilização dentro da linguagem vai muito além).

Exemplo: 

      <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
      </ul>

      Será exibido:
  • Item 1
  • Item 2
  • Item 3

Links

(Voltar)

A linguagem HTML incorpora mais dinamismo à apresentação das informações quando nos apropriamos dos links que podem ser utilizados para interligar diferentes pontos da página, assim como vincular o conteúdo atual a uma página de outro arquivo. Esta tag trabalha com o atributo href que armazena o endereço externo ao qual está vinculado.

Exemplo: 

      <a href="https://www.google.com.br">Este é um link para o Google</a>
      
      Será exibido:
Este é um link para o Google
